# 1.2 XE.Progress
Progress bar의 생성, 종료를 관리하는 오브젝트입니다. XE ajax사용시 기본설정으로 나타나게 되어 있습니다.

## XE.Progress.start()
Progress bar를 생성하는 UI를 화면에 노출합니다. 
```javascript
XE.Progress.start(); // progress bar 생성
```
## XE.Progress.done()
Progress bar를 생성하는 UI를 종료합니다.
```javascript
XE.Progress.done(); // progress bar 제거
```
